# Project Context — Human Music Platform

This is an event-driven Web3 music streaming MVP.

The system currently supports:

- stream simulation
- event processing
- blockchain interaction (Algorand)
- frontend interaction via Next.js

---

# Current Architecture State

The system is NOT yet production-ready.

It is a structured MVP with:

- working backend
- working frontend
- working blockchain integration
- working event-driven system

---

# Core Goal

Evolve this MVP into a scalable music streaming platform.

---

# Key Principle

Blockchain must remain invisible to the user.

Frontend must behave like a normal streaming app.
